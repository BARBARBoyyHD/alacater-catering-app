## ADDED Requirements

### Requirement: Bottom Action Bar
The system SHALL display a bottom action bar with Chat and "Pesan Sekarang" buttons.

#### Scenario: Bottom action bar display
- **WHEN** user is on the product detail screen
- **THEN** system shows an action bar at the bottom with functional "Chat" and "Pesan Sekarang" buttons

### Requirement: Initiate Order
The system SHALL allow the user to click "Pesan Sekarang" to start the order process.

#### Scenario: Initiate order
- **WHEN** user clicks "Pesan Sekarang"
- **THEN** system navigates the user to the order screen
