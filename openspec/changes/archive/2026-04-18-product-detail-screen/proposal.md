## Why

To improve the user experience by enabling users to view detailed product information and initiate the ordering process, consistent with the project's design and requirements.

## What Changes

- Implement `ProductDetailScreen` in React Native, mirroring the UI from `docs/ui/ProductDetailScreen.tsx`.
- Include Hero Banner, Provider Info, Pricing & Options, "Tentang Paket" (expandable), Menu Carousel, Review Section, and Bottom Action Bar.

## Capabilities

### New Capabilities
- `product-detail-view`: Enables viewing detailed information for a specific product.
- `product-order-init`: Enables initiating the order process from the detail screen.

### Modified Capabilities

## Impact

- Adds new screen component to the mobile application.
- Updates navigation stack.
