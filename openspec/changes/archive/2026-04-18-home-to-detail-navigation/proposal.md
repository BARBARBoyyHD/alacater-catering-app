## Why
Users need to be able to navigate from the home screen to the product detail screen to view more information and initiate the ordering process.

## What Changes
- Implement navigation from `HomeScreen` to `ProductDetailScreen`.
- Use React Navigation's `navigation.navigate` to move between screens.

## Capabilities

### New Capabilities
- `home-to-detail-navigation`: Enables navigating from home screen items to the product detail screen.

### Modified Capabilities
- None.

## Impact
- Updates `HomeScreen.tsx` and `ProviderCard.tsx`.
- Requires navigation setup in the app's root or stack navigator.
